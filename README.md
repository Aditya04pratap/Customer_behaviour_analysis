# Data Analytics Project

## Overview

This project demonstrates an end-to-end data analytics workflow, from data collection to business insights. The project includes data preprocessing, exploratory data analysis (EDA), SQL-based analysis, interactive dashboard development, and reporting using industry-standard tools.

---

## Dataset

The dataset was analyzed to identify trends, patterns, and key insights. It contains structured data that was cleaned and transformed before analysis.

---

## Tools & Technologies

* **Python** (Pandas, NumPy, Matplotlib, Seaborn)
* **Jupyter Notebook**
* **PostgreSQL**
* **SQL**
* **Power BI**
* **Gamma** (Presentation)
* **Git & GitHub**

---

## Project Workflow

### 1. Data Import

* Loaded the dataset into Python.
* Performed an initial inspection of the data.

### 2. Exploratory Data Analysis (EDA)

* Analyzed data distributions and relationships.
* Identified missing values, duplicates, and outliers.
* Generated visualizations to understand trends.

### 3. Data Cleaning

* Removed duplicate records.
* Handled missing values.
* Corrected data types.
* Prepared the dataset for analysis.

### 4. SQL Analysis

* Imported the cleaned dataset into PostgreSQL.
* Wrote SQL queries to answer business questions.
* Performed aggregations, filtering, joins, and analytical queries.

### 5. Power BI Dashboard

* Connected Power BI to the processed dataset.
* Built an interactive dashboard with KPIs, charts, and filters.
* Designed visuals to communicate insights effectively.

### 6. Report & Presentation

* Documented findings in a project report.
* Created a presentation using Gamma summarizing the analysis and key insights.

## How to Run

1. Clone the repository.
2. Install the required Python libraries.
3. Open the Jupyter Notebook and run the EDA and data cleaning notebooks.
4. Import the cleaned dataset into PostgreSQL.
5. Execute the SQL queries.
6. Open the Power BI (.pbix) file to explore the dashboard.
7. Review the report and presentation for project findings.
